# dazhongdianpingSpider
大众点评最新爬虫demo，可以批量爬取店铺评论评分等信息

## get_data.py
该文件用来收集网页源码信息，一共需要三个参数：
- 店铺id
- 要爬取的页数
- 浏览器的Cookie（每换一个店铺，需要重新从浏览器复制一份Cookie）

## dazhongSpider.py
该文件用来分析网页信息，提取评论信息并保存到数据库中
